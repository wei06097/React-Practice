# React 登入系統

### 指令
```
npm run db
npm run server
npm run start

node
require('crypto').randomBytes(64).toString('hex')
```

### 設計
- [登入頁面設計](https://www.youtube.com/watch?v=R9RuTbJ7WVg)
- 前端使用 React Router 做成 SPA
- 後端使用 express 框架 
- 使用 json-server 作為存放帳號密碼的 DB (暫時性)
